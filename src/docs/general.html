General

File Operations

Please note, for this version of the software only FCS3.0 has been tested. Support for reading .fcs files is provided by the fcsparser Python package, so all limitations with regards to files being readable are derived from this package.

Opening a file

To open a file, please first choose an appropriate .fcs file stored on you computer. Following this step, you will be shown an overview of all the parameters that could be extracted from your file. Choose the parameters that you want to analyse. They will be the only parameters used for visualisation and clustering.

Saving a file to .csv

You can save a file to the .csv format using the Save to .csv option. You will be asked to choose a location on your computer to which the file should be saved. You do not have to add the .csv file ending when typing the file name. Please note that you cannot use any other file endings when using this save option.
Please note that all parameters available in the original file will be saved to the .csv file regardless of your selection. In case any outliers have been removed during the clustering, these will not be saved to the new .csv file.
